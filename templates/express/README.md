# Express starter template

A minimal Express API server - copy this directory out to start a new
project.

## Recommended structure

```text
express-app/
├── src/
│   └── index.js
├── package.json
├── .env.example
├── .gitignore
├── .editorconfig
└── LICENSE
```

## Getting started

```bash
pnpm install
pnpm start
```

Then visit <http://localhost:3000>.

## Example configuration

Copy `.env.example` to `.env` for local config (`PORT`, etc.) - `.env` is
gitignored.
