# React starter template (Vite)

A minimal React + Vite starter - copy this directory out to start a new
project.

## Recommended structure

```text
react-app/
├── src/
│   ├── main.jsx
│   └── App.jsx
├── index.html
├── package.json
├── vite.config.js
├── .gitignore
├── .editorconfig
└── LICENSE
```

## Getting started

```bash
pnpm install
pnpm dev
```

## Example configuration

`vite.config.js` wires up `@vitejs/plugin-react`; add a proxy, aliases, or
env handling there as your project grows.
