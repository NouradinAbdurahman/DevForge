# Next.js starter template

A minimal Next.js (App Router, TypeScript) starter - copy this directory out
to start a new project.

## Recommended structure

```text
nextjs-app/
├── app/
│   ├── layout.tsx
│   └── page.tsx
├── package.json
├── tsconfig.json
├── next.config.mjs
├── .gitignore
├── .editorconfig
└── LICENSE
```

## Getting started

```bash
pnpm install
pnpm dev
```

Then open <http://localhost:3000>.

## Example configuration

`next.config.mjs` is intentionally empty (defaults); add config as your
project needs it. `tsconfig.json` uses Next's recommended strict settings.
