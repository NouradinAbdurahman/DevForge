# Minimal profile

Bare-essentials CLI tooling for any developer: git, GitHub CLI, Git LFS,
mise, jq/yq, fzf, ripgrep, GNU coreutils/sed/findutils, wget, tree. No
casks, no databases. Editor settings/keybindings/extensions are still
restored in full regardless of profile (see [../../docs/Profiles.md](../../docs/Profiles.md)).

Use with:

```bash
./bootstrap.sh --minimal
# or
./devforgekit install --profile minimal
```
